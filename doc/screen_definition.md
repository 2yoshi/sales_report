## 2. SCR-001 ログイン画面

### 概要
システムへのログイン画面。メールアドレスとパスワードで認証する。

### アクセス
- URL: `/login`
- 認証不要

### 画面レイアウト

```
┌─────────────────────────────────────┐
│                                     │
│         営業日報システム             │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ メールアドレス                │  │
│  │ [                           ] │  │
│  │                               │  │
│  │ パスワード                    │  │
│  │ [                           ] │  │
│  │                               │  │
│  │  [     ログイン     ]         │  │
│  └───────────────────────────────┘  │
│                                     │
└─────────────────────────────────────┘
```

### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|----------------|
| メールアドレス | text (email) | ○ | メール形式 |
| パスワード | password | ○ | 8文字以上 |

### ボタン・アクション

| ボタン | アクション |
|--------|-----------|
| ログイン | 認証処理 → 成功時はSCR-002へ遷移 |

### エラー表示
- 認証失敗: 「メールアドレスまたはパスワードが正しくありません」

## 3. SCR-002 ダッシュボード（日報一覧）

### 概要
ログイン後のトップページ。営業は自分の日報一覧、上長は部下全員の日報一覧を確認できる。

### アクセス
- URL: `/`
- 認証必要: 全ロール

### 画面レイアウト

```
┌─────────────────────────────────────────────────────────┐
│ [ロゴ] 営業日報システム        [山田 太郎] [ログアウト] │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  日報一覧                        [+ 新規作成]          │
│                                                         │
│  絞り込み: [担当者 ▼] [期間: ____〜____] [検索]        │
│                                                         │
│  ┌───────┬──────────┬───────────┬────────┬──────────┐  │
│  │ 日付  │ 担当者   │ 訪問先    │ステータス│ 操作   │  │
│  ├───────┼──────────┼───────────┼────────┼──────────┤  │
│  │04/19  │ 山田 太郎│ A社、B社  │ 未コメント│ [詳細]│  │
│  │04/18  │ 山田 太郎│ C社       │ コメント済│ [詳細]│  │
│  │04/17  │ 鈴木 一郎│ D社、E社  │ 未コメント│ [詳細]│  │
│  └───────┴──────────┴───────────┴────────┴──────────┘  │
│                                                         │
│  < 前へ  1 / 5  次へ >                                  │
└─────────────────────────────────────────────────────────┘
```

### 表示項目

| 列名 | 内容 |
|------|------|
| 日付 | 日報の対象日 |
| 担当者 | 日報作成者の氏名（上長には全員分表示） |
| 訪問先 | 訪問した顧客名（複数の場合はカンマ区切り、3社以上は「他N社」） |
| ステータス | コメント済 / 未コメント |
| 操作 | 詳細ボタン（SCR-004へ遷移） |

### フィルタ

| 項目 | 内容 |
|------|------|
| 担当者 | 営業は自分のみ表示。上長はプルダウンで部下を絞り込み可能 |
| 期間 | 開始日・終了日を指定（デフォルト: 直近30日） |

### ボタン・アクション

| ボタン | 表示条件 | アクション |
|--------|---------|-----------|
| 新規作成 | 営業ロールのみ | SCR-003へ遷移 |
| 詳細 | 全員 | SCR-004へ遷移 |

## 4. SCR-003 日報作成画面

### 概要
営業担当者が当日の日報を新規作成する画面。訪問記録を複数行追加できる。

### アクセス
- URL: `/reports/new`
- 認証必要: 営業ロールのみ

### 画面レイアウト

```
┌───────────────────────────────────────────────────────┐
│ 日報作成                                              │
│                                                       │
│  対象日: [2026-04-19]                                 │
│                                                       │
│ ── 訪問記録 ──────────────────────────────────────── │
│                                                       │
│  ┌────────────────────────────────────────────┐      │
│  │ 顧客 [        顧客を選択 ▼       ]  [削除] │      │
│  │ 訪問内容                                   │      │
│  │ [                                        ] │      │
│  │ [                                        ] │      │
│  └────────────────────────────────────────────┘      │
│                                                       │
│  ┌────────────────────────────────────────────┐      │
│  │ 顧客 [        顧客を選択 ▼       ]  [削除] │      │
│  │ 訪問内容                                   │      │
│  │ [                                        ] │      │
│  └────────────────────────────────────────────┘      │
│                                                       │
│  [+ 訪問先を追加]                                     │
│                                                       │
│ ── Problem（課題・相談）──────────────────────────── │
│  [                                                  ] │
│  [                                                  ] │
│                                                       │
│ ── Plan（明日やること）───────────────────────────── │
│  [                                                  ] │
│  [                                                  ] │
│                                                       │
│          [キャンセル]  [下書き保存]  [提出]           │
└───────────────────────────────────────────────────────┘
```

### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|----------------|
| 対象日 | date | ○ | 当日以前の日付。同じ担当者・日付の日報が既存の場合エラー |
| 顧客（訪問記録） | select | ○ | 顧客マスタから選択。訪問記録が1件以上必要 |
| 訪問内容 | textarea | ○ | 最大1000文字 |
| Problem | textarea | ○ | 最大2000文字 |
| Plan | textarea | ○ | 最大2000文字 |

### ボタン・アクション

| ボタン | アクション |
|--------|-----------|
| 訪問先を追加 | 訪問記録行を1行追加（上限なし） |
| 削除 | 該当の訪問記録行を削除（1行のみの場合は非表示） |
| キャンセル | SCR-002へ戻る（確認ダイアログ表示） |
| 下書き保存 | 未提出状態で保存。一覧から再編集可能 |
| 提出 | バリデーション後、日報を提出。SCR-002へリダイレクト |

## 5. SCR-004 日報詳細・編集画面

### 概要
日報の内容を閲覧・編集する画面。上長はコメントを追加できる。営業は自分の未提出日報を編集できる。

### アクセス
- URL: `/reports/:id`
- 認証必要: 全ロール

### 画面レイアウト

```
┌──────────────────────────────────────────────────────────┐
│ 日報詳細                [← 一覧へ]  [編集]  [削除]      │
│                                                          │
│  担当者: 山田 太郎       対象日: 2026-04-19              │
│                                                          │
│ ── 訪問記録 ─────────────────────────────────────────── │
│                                                          │
│  1. 顧客: 株式会社A                                      │
│     訪問内容: 定例ミーティング。新製品の提案を実施。      │
│              予算感について確認し、前向きな回答を得た。  │
│                                                          │
│  2. 顧客: 有限会社B                                      │
│     訪問内容: 契約更新の打ち合わせ。来週中に回答予定。   │
│                                                          │
│ ── Problem（課題・相談） ─────────────────────────────── │
│  株式会社Aの予算承認フローが不明。意思決定者に直接        │
│  アプローチする方法を相談したい。                        │
│                                                          │
│ ── Plan（明日やること） ──────────────────────────────── │
│  ・株式会社Aの担当者にフォローアップメールを送付         │
│  ・有限会社Bの契約書ドラフトを作成                      │
│                                                          │
│ ── コメント ──────────────────────────────────────────── │
│                                                          │
│  ┌────────────────────────────────────────────────┐    │
│  │ 田中 部長  2026-04-19 18:32                    │    │
│  │ 株式会社Aは購買部門の鈴木さんが決裁権限を持って │    │
│  │ います。明日の訪問で紹介してもらいましょう。    │    │
│  └────────────────────────────────────────────────┘    │
│                                                          │
│  コメントを追加（上長のみ）                              │
│  [                                                     ] │
│  [                                                     ] │
│                        [コメントを投稿]                  │
└──────────────────────────────────────────────────────────┘
```

### 表示項目

| セクション | 内容 |
|----------|------|
| ヘッダー | 担当者氏名・対象日 |
| 訪問記録 | 顧客名・訪問内容（登録順に連番表示） |
| Problem | 課題・相談テキスト |
| Plan | 明日やることテキスト |
| コメント | 投稿者名・投稿日時・コメント本文（新しい順） |

### 編集・コメント権限

| アクション | 営業（本人） | 営業（他人） | 上長 |
|-----------|-------------|------------|------|
| 閲覧 | ○ | △（部下のみ） | ○（部下のみ） |
| 編集 | ○（未提出のみ） | ✕ | ✕ |
| 削除 | ○（未提出のみ） | ✕ | ✕ |
| コメント投稿 | ✕ | ✕ | ○ |

### ボタン・アクション

| ボタン | 表示条件 | アクション |
|--------|---------|-----------|
| ← 一覧へ | 全員 | SCR-002へ戻る |
| 編集 | 本人かつ未提出 | 項目を編集モードに切り替え |
| 削除 | 本人かつ未提出 | 確認ダイアログ → 削除後SCR-002へ |
| コメントを投稿 | 上長のみ | コメントを保存してページ内に追加表示 |

## 9. SCR-008 営業マスタ登録・編集（管理者専用）

### 概要
ユーザー（営業・上長）の新規登録または編集を行う画面。

### アクセス
- URL: `/users/new`（新規）、`/users/:id/edit`（編集）
- 認証必要: 管理者ロールのみ

### 画面レイアウト

```
┌────────────────────────────────────────────┐
│ ユーザー登録                               │
│                                            │
│  氏名 *                                    │
│  [                                       ] │
│                                            │
│  メールアドレス *                          │
│  [                                       ] │
│                                            │
│  パスワード *（新規時のみ）                │
│  [                                       ] │
│                                            │
│  ロール *                                  │
│  ○ 営業  ○ 上長  ○ 管理者               │
│                                            │
│       [キャンセル]  [保存]                 │
└────────────────────────────────────────────┘
```

### 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|----------------|
| 氏名 | text | ○ | 最大100文字 |
| メールアドレス | text (email) | ○ | メール形式、システム内一意 |
| パスワード | password | ○（新規のみ） | 8文字以上 |
| ロール | radio | ○ | 営業 / 上長 / 管理者 のいずれか |

### ボタン・アクション

| ボタン | アクション |
|--------|-----------|
| キャンセル | SCR-007へ戻る |
